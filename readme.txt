📚 Student Study Center Management System
A comprehensive and user-friendly application designed to manage the daily operations of a student study center efficiently. This system streamlines student registration, session tracking, attendance management, course enrollment, and more, providing both administrators and students with a centralized platform.

🚀 Features
🧑‍🎓 Student Registration & Profile Management

📝 Course & Subject Allocation

🕒 Session Scheduling & Attendance Tracking

📈 Performance & Progress Reports

🧑‍🏫 Admin Dashboard for Center Management

🔒 Secure Login System for Students and Admins

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript

Backend: PHP

Database: MySQL

Tools: XAMPP / WAMP

📌 Installation & Setup
Clone the repository

git clone https://github.com/your-username/student-study-center-management.git
Setup Database

Open phpMyAdmin in your local server.

Import the provided SQL file (study_center.sql).

Configure the Project

Move the project folder to htdocs (XAMPP) or www (WAMP).

Edit db_config.php with your local database credentials.

Run the Project
🔗 [Admin Panel - Localhost Access](http://localhost/sscms/sscms/admin/)


How to run the Student Study Center Management System (SSCMS) Project

1. Download the zip file

2. Extract the file and copy sscms folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with the name sscmsdb

6. Import sscmsdb.sql file(given inside the zip package in the SQL file folder)

7. Run the script http://localhost/sscms
